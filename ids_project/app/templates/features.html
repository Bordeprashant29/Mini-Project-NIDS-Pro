🔍 Key Features of NIDS Pro

Explore the powerful tools and technologies integrated into NIDS Pro. From real-time packet analysis to ML-driven detection, our system is designed to provide comprehensive, automated intrusion detection.

🧠 Machine Learning Detection

NIDS Pro leverages a Random Forest Classifier trained on NSL-KDD to identify both known and unknown attacks, reducing false positives and improving detection accuracy.

📦 Preprocessing Pipeline

Integrated data preprocessing with custom encoding and scaling ensures all categorical and numerical data is formatted optimally before prediction.

📊 Interactive Dashboard

Upload datasets, get instant predictions, view result summaries, and download full reports—all through a modern, responsive interface.

🔐 User Authentication

Secure registration and login system with admin support, username-based access, and session control.

⚠️ Real-Time Analysis

Processes data immediately upon upload to identify suspicious patterns and anomalies in real-time.

📁 CSV Format Support

Supports standardized 41/42/43-feature NSL-KDD-style CSVs with built-in error handling and format validation.

📥 Report Download

After analysis, users can download the full result CSV which includes predictions and preserved original features.

🎯 Accuracy Optimized

Trained with class_weight="balanced" to handle class imbalance and ensure strong generalization to unseen attack types.
