About NIDS Pro

NIDS Pro is an AI-powered Network Intrusion Detection System designed to detect malicious network activity in real-time using advanced machine learning models. It provides users with intuitive tools to analyze, predict, and respond to security threats with speed and accuracy.

üîí What is NIDS?

A Network Intrusion Detection System monitors traffic flowing across the network. It detects suspicious activities by comparing network packets against known attack signatures or through behavioral analysis.

üöÄ How It Works

  1. The user logs in and uploads a CSV of captured network activity.
  2. Preprocessing is applied (encoding, scaling) via a trained ML pipeline.
  3. The system predicts each row as Normal or Attack.
  4. Results are shown on screen and made available as downloadable CSV.

  • static/ - CSS, image assets, and prediction reports.

  • utils.py - Utility functions like loading the trained model.

üß† Machine Learning Model

The backend ML model is a Random Forest Classifier trained on the NSL-KDD (20% sample) dataset. The model includes:

  • Custom categorical encoding for protocol, service, and flags.
  • Standardization via StandardScaler.
  • Class balancing using class_weight="balanced".
  • Saved with joblib for efficient loading and inference.

üìä Input File Format

Your CSV file should contain 41 feature columns, followed optionally by a label and/or a final column like count. Columns should include:

duration, protocol_type, service, flag, src_bytes, dst_bytes, ..., dst_host_srv_rerror_rate
